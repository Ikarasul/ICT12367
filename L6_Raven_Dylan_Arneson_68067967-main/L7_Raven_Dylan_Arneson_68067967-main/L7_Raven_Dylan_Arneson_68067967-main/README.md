โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา ICT12367 การใช้กรอบงานสำหรับการพัฒนาเว็บแอปพลิเคชัน เพื่อความมั่นคงปลอดภัย โดยมีวัตถุประสงค์เพื่อฝึกฝนการสร้างโปรเจกต์ด้วย Django Framework รวมถึงการจัดการเส้นทาง (URL Routing) และการแสดงผล (View)

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

* **Python & Django Framework:** ใช้เป็นโครงสร้างหลักในการสร้างเว็บแอปพลิเคชัน (Web Application) และจัดการเส้นทาง URL
* **HTML:** ใช้สำหรับการสร้างเทมเพลต (Template) เพื่อแสดงผลหน้าเว็บและแบบฟอร์มบันทึกข้อมูล
* **SQLite3:** ฐานข้อมูลที่ถูกสร้างขึ้นมาเป็นค่าเริ่มต้นเมื่อเริ่มโปรเจกต์ Django (`db.sqlite3`)

## ✨ คุณสมบัติหลักและการทำงาน (Features)

โปรเจกต์นี้ได้กำหนด URL Patterns ในไฟล์ `urls.py` และสร้างฟังก์ชันเพื่อตอบสนองการร้องขอของผู้ใช้ในไฟล์ `views.py` ดังนี้:

* **หน้าหลัก (Index):** เมื่อเข้าสู่ URL `/` จะแสดงข้อความ "หน้าแรกของเว็บไซต์วิชา ICT12367"
* **หน้าเกี่ยวกับเรา (About):** เมื่อเข้าสู่ URL `/about` จะแสดงข้อความ "เกี่ยวกับเรา"
* **หน้าแบบฟอร์ม (Form):** เมื่อเข้าสู่ URL `/form/` ระบบจะเปลี่ยนจากการใช้ `HttpResponse()` มาใช้คำสั่ง `render()` เพื่อโหลดไฟล์เทมเพลต `form.html` ซึ่งเป็นแบบฟอร์มบันทึกข้อมูลพนักงานมาแสดงผล
* **หน้าติดต่อ (Contact):** เมื่อเข้าสู่ URL `/contact/` จะเรียกใช้งานฟังก์ชัน `contact(request)` เพื่อแสดงข้อความ "ติดต่อ" ตามด้วยรหัสนักศึกษา ชื่อ และนามสกุล โดยใช้แท็ก `<h1>`
