## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

* **HTML5:** โครงสร้างหลักของหน้าเว็บ
* **Bootstrap 5 (CDN):** เฟรมเวิร์กสำหรับจัด Layout (Grid System) และทำ Responsive
* **CSS3:** ใช้สำหรับการตกแต่งเพิ่มเติม เช่น
    * Hover Effects (เอฟเฟกต์เมื่อนำเมาส์ไปชี้)
    * Transitions (การเปลี่ยนผ่านที่นุ่มนวล)
    * Box-shadow (การใส่เงาให้วัตถุ)

---

## ✨ คุณสมบัติหลัก (Features)

* **📱 Responsive Design:** รองรับการแสดงผลได้อย่างสมบูรณ์ทั้งบนมือถือ (Mobile) และเดสก์ท็อป (Desktop)
* **🖱️ Hover Effects:** เพิ่มลูกเล่นเมื่อนำเมาส์ไปชี้ที่รูปภาพ รูปจะทำการขยาย (Zoom) และมีเงาปรากฏขึ้น
* **nav Navbar:** แถบเมนูนำทางด้านบนที่ปรับเปลี่ยนรูปแบบตามขนาดหน้าจออัตโนมัติ
